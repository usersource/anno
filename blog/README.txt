To setup or update the blog:
- refer to https://googlecloudplatform.github.io/appengine-php-wordpress-starter-project/
- download the appropriate build (windows/Linux/mac)
- extract into the folder blog
- overwrite the subset of extracted files with those from our repo 
  - app.yaml, cron.yaml, php.ini, wordpress/wp-config.php
  - also, obtain the commited themes and plugins (if any) from our repo - these will go into wordpress/wp-content/themes

- on the server
- setup permalinks (settings->general->permalinks)
- activate the appropriate theme
- customize the theme (remove widgets. upload images etc)
